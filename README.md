# react-native-reanimated-tooltip

Tooltip for React Native using React Native Reanimated and Modal

## Installation

```sh
npm install react-native-reanimated-tooltip
```

## Usage

```js
import { Tooltip } from 'react-native-reanimated-tooltip';

```
